Hub-Camp

En el programa de Desarrollo Front-End te enseñaremos a crear experiencias de usuario con Tecnología Web. Con nuestro Hub-Camp dominarás una colección de tecnologías: HTML, CSS, JavaScript, Frameworks, entre otras herramientas para el diseño e implementación de UI/UX poderosamente interactivas.

Nuestro Programa

Ambiente de trabajo

Conocerás todas las herramientas que necesitamos para poder empezar a desarrollar.

Fundamentos de Desarrollo Web

Comprenderás el perfil de un desarrollador web front-end, tareas e impacto en el mundo web.

HTML

Dominarás la herramienta más básica para crear sitios web, en su versión más reciente; su sintaxis y mejores prácticas de uso.

Gestión de versiones

Aprederás la importancia a almacenar las versiones de tu código, y los comandos necesarios para poder hacerlo.

CSS

Reconocerás las formas de incluir estilos a tu página web, en su versión más reciente; su sintaxis y mejores prácticas de uso.

Responsive Design

Practicarás la forma de adaptar la apariencia de tu página web a los diferentes dispositivos

Diseño UI/UX

Emplearás algunas prácticas en tu desarrollo para garantizar una buena experiencia de tus usuarios que interactuan con tu página web.

Fundamentos de programación

Asimilarás los fundamentos, las reglas y nociones básicas para aprender a programar.

JavaScript

Practicarás el lenguaje de programación, mejor conocido para las páginas web en su versión más reciente; su sintaxis y mejores prácticas de uso.

Frameworks

Aprenderás a estandarizar prácticas y conceptos de tu marco de trabajo.

Metodologías de desarrollo de software

Sabrás como estructurar, planear y controlar el proceso de desarrollo de software.
